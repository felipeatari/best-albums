Sobre a Aplicação

Uma visão geral

Está simples aplicação tem intuito de servir como uma espécie de guia onde é reunido os melhores albums (cd's) das mais variadas bandas de rock, e que contém a informação do nome da banda, ano de lançamento e as suas respetivas faixas e ainda conta com a imagem que representa a capa do album.

Sobre o Front-End

Já no Front-End as os dados são enviados e recebidos do servidor através de requisições Ajax, onde é utilizado o fetch API do JavaScript. E para estruturação das páginas claro que foi criada com HTML e CSS para estiliza-las.

Sobre o Back-End

Essa aplicação Web apesar de simples, foi desenvolvida com as melhores práticas de programação, onde foi utilizado o padrão de Arquitetura de Software MVC, e utilizado PHP Orientado a Objetos no Back-End. Essa aplicação também conta com um gernciador de rotas, uma API própria e um renderizador de páginas, tudo criado com PHP puro.

Sobre a Manipulação de Dados

Foi utilizado o banco de dados MySql para tratar todos os dados que essa aplicação for precisar manipular, onde contém um CRUD que respeita os verbos HTTP (GET, POST, PUT e DELETE).

Considerações Finais

Como mencionado anteriormente, o projeto é simples, porém pode ser expandido ou feito qualquer outra coisa com ele, pois está bem estruturado. Não foi utilizado nenhum Framework ou biblioteca, foi criado todo na mão mesmo. O código fonte está disponível no meu Github.
